Maven

Maven or Apache Maven is an open source tool to build and maintain the projects. It's primarily used for Java projects but it can also be used for projects written in C#, Ruby, Scala and many other languages.

Build:

Maven is a build-automation tool which will help us to handle the build lifecycles by handling the dependencies. Below are the default lifecycle of Maven:

  • Validate
  • Compile
  • Test
  • Package
  • Install
  • Deploy

Dependency Management:

A predominant feature in Maven is dependency management. Maven has a central repository for all commonly used libraries and plugins. All the dependent tools and plugins of the project are tracked in POM file with reference to central maven repository. When project is build for first time, all the dependencies will be downloaded from central repository and added to local repository. If there is any transit dependencies, then they are automatically handled by Maven which gives big relief to the developers.

Managing Project Structure:

Maven by default creates the folder hierarchy for managing the project files. This helps to manage/maintain the project files efficiently.
