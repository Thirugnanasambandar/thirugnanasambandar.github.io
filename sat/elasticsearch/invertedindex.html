Inverted Index

Inverted index is a datastructure to stores the documents for faster and effecient retrival of data based on keywords search. Say for example we want to retrieve all the documents containing thee keyword 'Pizza', elasticsearch would work perfectly for such scenarios. It is a most popular data structure used in document retrival systems like search engine. Its very dynamic and has less latency to make a document available for testing from the time of adding document.

How Does It Work?

Inverted index parses each document and creates the indexes of each word and their occurence in the documents.
Consider the below sentences: 

            
    
                
  • Document 1: Car is a four wheeler
    • 
                
  • Document 2: Jeep has four wheel drive
    • 
                
  • Document 3: Car is available in both automatic and manual drive modes
    • 
                
  • Document 4: Jeep is available only in manual drive mode
    • 
                
  • Document 5: Car enthusiasts prefer manual dirve mode cars
    •

Each docment is parsed and a posting list will be created with each keyword as key and the list of document ids in which the keyword is appearing as value like below:

Car (1, 3, 4)
is (1, 3, 4)

both(3)

Using this approach, we have create a data dictionary for all the keywords in documents.

Steps to Creat Inverted Index:

Above explanation is just high level idea, building an inverted index would need below steps to match the strings correctly:
  • Parse the document: Parse the document and identify unique words.
  • Remove stop words: Stop words are most occuring and useless words in document like "a", "an", "I", "we, "then", "has", "have", etc.
  • Stemming: Identify the root words for plural words(cars will be treated as car), adjectives and tenses(ran will be converted to run)
  • Add the words to data dictionary: Final list of words will be added to data dictionary with entry for the document id

Final data structure for the above documents might look like below after following all the steps:
