Jenkins

Jenkins is a famous tool for CI-CD. CI stands for Continous Inegration where as CD stands for Continous Delivery and Continous Deployment.
  • CI - Continous Integration: Continous Integration is the practice of testing the code in a common server everytime when code is commited into the source repository.
  • CD - Continous Deployment: Preparation process for making the code to be deployable
  • CD - Continuous Deployment: is the process of deployment
