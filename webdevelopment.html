RGBA

Transpareny & Opacity of back-ground color property can be controlled using the method rgba(red, green, blue, alpha). This property valuue doesn't font and other overlapping elements but only the bg color of cuurrent element.
  • Red/Green/Blue Values ranges from 0-255
  • Alpha value ranges from 0 to 1. Eg: 0 is 0% opactiy. 1 is 100% opacity
Below example for opacity propery. Outer box is 'section' and inner boxes are 'div'.
  • Div1 is having Alpha value as '1' (i.e. 0% transparency/100% opacity)
  • Div2 is having Alpha value as '0' (i.e. 100% transparency/0% opacity)
  • Div3 is having Alpha value as '.7' (i.e. 30% transparency/70% opacity)

Opacity

Opacity of the container, its content and decendents can be controlled using 'opacity' property. Opacity values range from 0 to 1. 0 is 0% opacity or 100% transparent whereas 1 is 100% opacity or 0% transparent. RGBA doesn't affect the content or descendants but opacity afftects both.

  • Div3 is having opacity value as '.7' (i.e. 30% transparency/70% opacity)

Position

Position property sets how an element is positioned in a document. The top, right, bottom and left properties determine the final location of positioned elements. List of allowed values for 'position' property are:

  • static: Element is positioned according to normal flow of the document. The top, right, bottom and left properties have no effect.
  • relative: Element is positioned according to normal flow of the document and then offset relative to itself based on the values of top, right, bottom and left. Space is created for the element as if its in static mode.
  • absolute: Element is removed from the normal document flow and no space is created in for the element in the page layout. Its positioned relative to its closest ancestor, if any; otherwise, its placed relative to the initial containing block.
  • fixed: Element is removed from the normal document flow and no space is created in for the element in the page layout. The element is position relative to the initial containing block, except when one of its ancestors has a transform, perspective or filter property set.
